# 🧪 Chemical World / 化学世界

**Not magic. Chemistry.**  
From a beaker to a rocket. Real reactions. Real machines. Real space.

> 不是魔法，是化学。从烧杯到火箭。真实反应、真实机器、真实太空。

## 🎮 What is this? / 这是什么？

A hardcore chemistry + industrial crafting game.  
Combine elements. Earn coins. Buy machines. Build rockets. Leave the planet.

一个硬核化学合成 + 工业制造游戏。  
合成物质 → 赚取硬币 → 购买机器 → 组装火箭 → 离开地球。
